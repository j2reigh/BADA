# BADA Career Direction Report - Product Design

## Core Concept (v3)

> **"운을 확인하지 말고, 설계하라"**
> **"Stop Checking Your Fortune — Design Your Luck"**

### 철학적 기반: "Design Your Luck" 프레임
(출처: designyourluck 영상 분석)

**핵심 통찰:**
- 운(運)의 한자 뜻 = "움직임", "길"
- Career의 어원 = 라틴어 carraria (마차 길, 경주로)
- **운은 복권이 아니라 도로다** — 도박이 아니라 건축의 문제

> "Luck is the cumulative momentum of the path you have already walked."
> "운은 이미 걸어온 길의 누적된 모멘텀이다."

### BADA Career의 포지셔닝
| 기존 운세 앱 | BADA Career |
|-------------|-------------|
| "올해 운이 좋을까?" (수동적 확인) | "올해 어떻게 움직여야 하나?" (능동적 설계) |
| Fortune Checking | Luck Architecture |
| 운이 나에게 뭘 해줄까 | 내가 운을 어떻게 만들까 |

### 핵심 질문 3개
1. **AI가 대체 못하는 나만의 역량은?** (불대체성)
2. **지금 어느 계절이고, 이 계절의 엑싯 전략은?** (타이밍 + 전략)
3. **이 회사가 아니라면 어떤 방향?** (Plan B)

### Input 전략: 산업/직무 묻지 않는다

**이유:**
- "마케터"라고 해도 실제로 하는 일은 천차만별
- 유저가 입력한 직무 ≠ 실제 역량
- 잘못된 input은 잘못된 output

**대안: 역량에서 역산**
- 사주/HD 기반 "타고난 역량" 먼저 진단
- 그 역량이 빛나는 **역할 유형** 제안 (직무명이 아님)
- 유저가 스스로 "아, 내가 지금 하는 일이 이거랑 맞네/안 맞네" 판단

**예시:**
```
❌ 기존 방식:
Input: "저는 IT 회사 마케터입니다"
Output: "마케터로서 이렇게 하세요"

✅ BADA 방식:
Input: 생년월일/시간/장소만
Output: "당신의 불대체 역량은 '품질 판단 + 설득력 있는 표현'입니다.
       이 역량이 빛나는 역할: 브랜드 가디언, 에디터, 투자 심사역
       → 지금 하는 일이 이 역할과 얼마나 겹치는지 스스로 판단하세요"
```

## 0. AI 불대체성 프레임워크

### AI가 못하는 것 (2026 기준)
| AI 약점 | 인간 역량 | 사주 데이터 매핑 |
|---------|----------|-----------------|
| 맥락 없는 창의성 | **독창적 조합력** | 상관(傷官) — 기존 규칙 파괴, 새로운 표현 |
| 관계 깊이 | **신뢰 기반 연결** | 정관(正官) — 조직 내 신뢰, 정재(正財) — 안정적 거래 |
| 모호한 상황 판단 | **직관적 의사결정** | 편관(偏官) — 위기 대응, 편인(偏印) — 통찰 |
| 물리적 현존 | **현장 장악력** | 비견(比肩) — 동료와 함께, 겁재(劫財) — 경쟁 속 생존 |
| 감정적 공명 | **정서적 교감** | 정인(正印) — 돌봄, 식신(食神) — 표현/예술 |

### 리포트 핵심 아웃풋
1. **Your Irreplaceable Edge** — AI가 대체 못하는 당신만의 역량 1-3개
2. **Optimal Environment** — 그 역량이 가장 빛나는 환경/산업/역할
3. **Career Timing** — 전환하기 좋은 시기 (대운/세운 기반)
4. **Plan B Blueprint** — "이 회사가 아니라면" 시나리오

## 0.5 Human Design 통합 — 정밀화 레이어

### 왜 Human Design인가?
- 사주: **"어떤 에너지를 타고났나"** (동양, 시간 기반)
- HD: **"어떻게 의사결정하고 행동해야 하나"** (서양, 뉴트리노 기반)
- 두 시스템이 **같은 방향**을 가리키면 → 확신도 ↑
- 두 시스템이 **다른 방향**을 가리키면 → 숨겨진 긴장 발견

### HD 데이터 → AI 불대체성 매핑

#### Type (에너지 운용 방식)
| HD Type | AI 불대체 역량 | 사주 대응 |
|---------|---------------|----------|
| **Manifesting Generator** | 빠른 피벗 + 다중 트랙 실행 | 식신+상관 조합 (다재다능) |
| Generator | 지속적 에너지 투입 | 비견+겁재 (체력, 끈기) |
| Projector | 시스템/사람 읽기 | 정관+편관 (관찰력) |
| Manifestor | 시작과 방향 설정 | 편관+상관 (독립적 추진) |
| Reflector | 환경 진단 | 정인+편인 (직관) |

#### Authority (의사결정 방식)
| Authority | 의미 | AI가 못 따라하는 이유 |
|-----------|------|---------------------|
| **Emotional** | 시간을 두고 감정 파동 확인 | AI는 즉답, 인간은 숙성 |
| Sacral | 즉각적 신체 반응 | AI는 몸이 없음 |
| Splenic | 순간 직관 | 설명 불가한 감각 |
| Ego | 의지력 기반 약속 | AI는 의지가 없음 |

#### Defined Centers (에너지 허브)
| Center | 정의됨(Defined) = 일관된 에너지 | 불대체 역량 |
|--------|-------------------------------|------------|
| **Throat** | 표현/커뮤니케이션 | 설득, 협상, 리더십 발화 |
| **Sacral** | 지속적 작업 에너지 | 마라톤 실행력 |
| **Solar Plexus** | 감정적 깊이 | 공감 기반 연결 |
| **Spleen** | 순간 직관, 건강 감각 | 위험 감지 |
| **Ajna** | 개념화, 분석 | 프레이밍 능력 |
| **Root** | 압박/스트레스 처리 | 데드라인 퍼포먼스 |
| Heart/Ego | 의지력, 약속 이행 | 신뢰 구축 |
| G Center | 방향, 정체성 | 비전 제시 |
| Head | 영감, 질문 | 문제 정의 |

#### Channels (에너지 흐름)
| Channel | 의미 | 불대체 역량 |
|---------|------|------------|
| **16-48 The Wavelength** | 깊이 있는 마스터리 추구 | 전문성 축적 (AI는 넓고 얕음) |
| **17-62 Acceptance** | 논리적 패턴 인식 + 표현 | 복잡한 것을 단순하게 설명 |
| **18-58 Judgment** | 교정/개선 본능 | 품질 판단 (AI는 기준이 없음) |
| **20-34 Charisma** | 즉각적 행동 + 현재 인식 | 현장 즉흥 대응력 |
| **35-36 Transitoriness** | 경험 추구 + 감정 깊이 | 스토리텔링, 경험 디자인 |

| 영역 | HD | 사주 | 종합 |

HD 18-58 채널과 사주 상관(傷官)이 같은 방향을 가리킵니다.

그리고 그 판단을 말과 글로 설득력 있게 전달하는 능력.

AI는 품질 기준을 스스로 만들지 못합니다.

⚠️ 주의:

## 1. Benchmark 분석: career-luck-chat의 강점

### 1.1 콘텐츠 구조 (5개 밀서)
| 밀서 | 핵심 가치 | BADA 활용 가능성 |
|------|----------|-----------------|
| 1밀서 | 기본 명반 + 2026년 기회/위기 | ✅ 연간 운세 |
| 2밀서 | 분기별 전략 + 전성기 경로 | ✅ 분기별 액션플랜 |
| 3밀서 | 생애 운세 그래프 + 위기-기회 매트릭스 | ✅✅ 시각화 핵심 |
| 4밀서 | TOP 5 황금기 + 연도별 점수 | ✅✅ 핵심 차별화 포인트 |
| 5밀서 | 인생 사계절 (봄/여름/가을/겨울) | ✅✅ 내러티브 프레임 |

### 1.2 가장 임팩트 있는 요소
1. **ASCII 그래프** — 텍스트 기반 시각화 (점수 곡선, 매트릭스)
2. **숫자화된 운세** — "90점", "82점" 등 점수로 표현
3. **시간축 프레임** — 분기별, 연도별, 10년 단위, 48년 사계절
4. **직설적 진단** — "준비 중독", "6월은 움직이지 마라", "봄에 심지 않은 씨앗은 가을에 거둘 수 없다"
5. **구체적 행동 지침** — "70% 준비되면 나가라", "12월에 정산하라"

### 1.3 핵심 가치 제안 (왜 19 USD?)
> "내 인생의 황금기가 언제인지 알고 준비하고 싶다"

- 기존 BADA (1.9 USD): "나는 어떤 사람인가" (정체성, 현재 상태)
- Career Report (19 USD): "내 황금기는 언제이고, 지금 뭘 해야 하나" (시간축, 미래 전략)

## 2. 제품 설계 원칙

### 2.1 빼야 할 것 (사주/점성술 용어)
| 원본 | 문제 | BADA 대안 |
|------|------|----------|
| "무토 일간", "갑오 대운", "자오충" | 이해 불가 | 행동 패턴 언어로 치환 |
| "정재", "편관", "식신" | 사주 전문용어 | "돈 버는 에너지", "압박 에너지", "실행 에너지" |
| "처녀자리 10하우스" | 점성술 용어 | 제거 또는 "커리어 변화 시기" |
| "생명수 7", "개인년수 1" | 수비학 용어 | "새 시작 에너지", "본질 발현 시기" |

### 2.2 남겨야 할 것 (알맹이)
1. **점수화된 시간축** — 연도별/분기별 점수 (0-100)
2. **생애 그래프** — ASCII 또는 SVG 시각화
3. **TOP 5 황금기** — 핵심 연도 하이라이트
4. **사계절 프레임** — 봄(준비), 여름(성장), 가을(수확), 겨울(정리)
5. **분기별 액션** — "언제 공격하고, 언제 방어하라"
6. **직설적 경고** — "6월에 결정하면 99% 후회한다"

## 3. 리포트 구조 설계 (v2 — AI 불대체성 중심)

### 3.1 페이지 구성
| 페이지 | 섹션 | 핵심 콘텐츠 |
|--------|------|------------|
| 1 | **Your Irreplaceable Edge** | AI가 대체 못하는 당신의 핵심 역량 1-3개 + 왜 이것인지 |
| 2 | **Your Optimal Zone** | 그 역량이 빛나는 환경/산업/역할 타입 |
| 3 | **Your Career Season** | 지금은 봄/여름/가을/겨울 중 어디인가 |
| 4 | **Lifetime Career Graph** | 20대-60대 점수 곡선 + TOP 3 황금기 |
| 5 | **This Year's Battlefield** | 올해 4분기 위기-기회 매트릭스 |
| 6 | **Plan B Blueprint** | "이 회사가 아니라면" 3가지 시나리오 |

### 3.2 사계절 × 독립 전략 (Conservative vs Aggressive)

**핵심 프레이밍: "회사 밖 독립"이 최종 목표**
- 이 리포트는 "더 좋은 회사 가기"가 아니라 **"언젠가 독립하기"**를 위한 것
- 같은 계절이어도 리스크 성향에 따라 전략이 다름
- 둘 다 보여주고 유저가 선택

#### 봄 (Spring) — 뿌리 내리기

| | Conservative 🐢 | Aggressive 🔥 |
|--|----------------|---------------|
| **전략** | 회사 다니면서 회사 돈으로 배워라 | 때려치고 뭘 좋아하는지 찾아라 |
| **구체적** | 사내 교육, 컨퍼런스, 자격증 — 전부 회사 비용으로. 퇴근 후 사이드 프로젝트. | 6개월~1년 갭이어. 여행, 다양한 경험, 저비용 실험. "뭘 하면 시간 가는 줄 모르나" 탐색. |
| **리스크** | 안전하지만 느림. 회사에 길들여질 수 있음. | 재정 압박. 방황이 길어질 수 있음. |
| **적합** | 비상금 6개월 미만. 부양가족 있음. | 비상금 1년 이상. 20대. 번아웃 상태. |

| | Conservative 🐢 | Aggressive 🔥 |
|--|----------------|---------------|
| **전략** | 성과 숫자화 → 이직으로 연봉/직급 점프 | 성과 숫자화 → 바로 독립 (프리랜서/1인 기업) |
| **구체적** | "나는 X를 Y% 성장시켰다" 포트폴리오 만들고, 연봉 30%+ 점프 이직. 새 회사에서 네트워크 확장. | 지금 하는 일을 프리랜서로 전환. 첫 클라이언트 = 전 회사 or 경쟁사. 바로 수익화. |
| **리스크** | 또 다른 회사에 종속. 독립 타이밍 놓칠 수 있음. | 수입 불안정. 영업/세금/계약 직접 해야 함. |
| **적합** | 아직 브랜드/네트워크 약함. 대기업 경력 필요. | 이미 업계에서 이름 있음. "나 없으면 안 돼" 상태. |

### 3.3 프랙탈 사이클 — 사주의 핵심 구조

**사주는 모든 걸 4단계로 본다 (生旺衰墓 / 봄여름가을겨울)**

같은 패턴이 스케일만 달라지면서 반복:

```
┌─────────────────────────────────────────────────────────┐
│                    48년 대주기                          │
│  ┌─────────┬─────────┬─────────┬─────────┐             │
│  │ 봄 12년 │ 여름12년│ 가을12년│ 겨울12년│             │
│  └─────────┴─────────┴─────────┴─────────┘             │
│                                                         │
│         ↓ 각 12년 안에서도 같은 구조 ↓                  │
│                                                         │
│  ┌─────────────────────────────────────────┐           │
│  │              12년 중주기                 │           │
│  │  ┌─────┬─────┬─────┬─────┐              │           │
│  │  │봄3년│여름3년│가을3년│겨울3년│              │           │
│  │  └─────┴─────┴─────┴─────┘              │           │
│  └─────────────────────────────────────────┘           │
│                                                         │
│         ↓ 각 1년 안에서도 같은 구조 ↓                   │
│                                                         │
│  ┌─────────────────────────────────────────┐           │
│  │              1년 소주기                  │           │
│  │  ┌─────┬─────┬─────┬─────┐              │           │
│  │  │Q1봄 │Q2여름│Q3가을│Q4겨울│              │           │
│  │  └─────┴─────┴─────┴─────┘              │           │
│  └─────────────────────────────────────────┘           │
└─────────────────────────────────────────────────────────┘
```

**리포트에서 3개 스케일 동시 제공:**

| 스케일 | 주기 | 용도 | 질문 |
|--------|------|------|------|
| **Macro** | 48년 (12년×4) | 인생 전체 조망 | "내 전성기는 언제?" |
| **Meso** | 12년 (3년×4) | 현재 대운 내 위치 | "이 시즌 안에서 지금은 어디?" |
| **Micro** | 1년 (분기×4) | 올해 전술 | "이번 분기에 뭘 해야?" |

**예시: 이지윤 (2026년)**

```
Macro (48년):
└─ 현재: 봄의 끝자락 (2016-2028)
   → 여름 시작까지 2년

Meso (12년 중 현재 3년 단위):
└─ 봄(2016-2028) 안에서:
   ├─ 봄의봄 (2016-2019): 씨앗 탐색 ✓ 지남
   ├─ 봄의여름 (2019-2022): 시행착오 ✓ 지남
   ├─ 봄의가을 (2022-2025): 방향 확정 ✓ 지남
   └─ 봄의겨울 (2025-2028): 뿌리 마무리 ★ 지금 여기
   → "봄의 겨울" = 봄 시즌 정리 + 여름 준비

Micro (2026년 분기):
└─ Q1: 봄 (칼 갈기)
   Q2: 여름 (6월 위험, 버티기)
   Q3: 가을 (9월 전력질주) ★ 핵심
   Q4: 겨울 (정산, 2027 설계)
```

**왜 이게 강력한가:**
1. **줌인/줌아웃** — 같은 프레임으로 다른 스케일을 봄
2. **구체적 행동** — "올해"뿐 아니라 "이번 3년", "이번 10년" 단위 전략
3. **일관성** — 사주 본래의 사이클 사고방식 그대로 활용
4. **안심** — "지금 겨울이라 힘든 거지, 봄이 온다" (어느 스케일에서든)

---

**리포트에서 보여줄 것:**
1. **Macro 진단** — 48년 중 어느 계절? 전성기까지 몇 년?
2. **Meso 진단** — 현재 12년 시즌 안에서 어느 3년 구간?
3. **Micro 진단** — 올해 4분기 중 어디가 핵심?
4. **Conservative vs Aggressive 전략** — 각 스케일별
5. 다음 전환점까지 남은 시간

## 7. 클로징 & CTA 전략

### "덕분에" 개념 — 운의 순환
(출처: designyourluck 영상 — 어머니의 화장실 이야기)

> 운은 순환이다. 흐름이 멈추면 운도 정체된다.
> Peak에 있을 때 시스템에 되돌려줘야 한다.
> "덕분에(Because of you)"를 표현하는 것이 다음 시즌의 운을 확보하는 방법.

### 클로징 멘트 (리포트 마지막)
```
운을 가장 빠르게 만드는 실천 방법:

이 리포트를 친구에게 공유하세요.

운(運)은 "움직임"입니다.
당신이 받은 인사이트를 다른 사람에게 전달할 때,
운의 흐름이 시작됩니다.

지금 머릿속에 떠오르는 한 사람에게 보내세요.
그것이 당신의 운을 설계하는 첫 번째 행동입니다.

[친구에게 공유하기]
```

### 왜 이 CTA가 강력한가
1. **이타적 프레이밍** — "나를 위해 공유"가 아니라 "운의 순환"
2. **즉각적 행동** — "지금 떠오르는 한 사람"
3. **철학적 정당화** — 운(運)의 어원과 연결
4. **바이럴 메커니즘** — 리포트가 퍼지면 BADA 인지도 상승

### 공유 플로우
1. 리포트 마지막에 CTA
2. "친구에게 공유하기" 버튼 → 친구 이름/생년월일 입력 폼
3. 친구에게 "OO님이 당신의 Career Report를 선물했습니다" 이메일
4. 친구는 무료 요약본 + 풀 리포트 할인 제안

## 8. 열린 질문

1. **점성술/수비학 제거 vs 단순화**
   - 완전 제거? → 사주만으로 점수 산출
   - 단순화? → "세 가지 운명 체계가 일치" 정도로 언급

2. **대운 정확도 문제**
   - 대운 순행/역행 계산에 출생시간 필요
   - 출생시간 모르는 유저 대응?

3. **연도별 점수 근거**
   - benchmark는 주관적 점수 (AI가 즉석에서 판단)
   - BADA는 일관된 알고리즘 기반 점수 필요

4. **경쟁사 분석 필요**
   - 사주아이 등 커리어 운세 제품 벤치마크
